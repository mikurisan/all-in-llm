理解现代智能体为何呈现出如今的形态, 以及其核心设计思想的由来, 本 section 将回溯其发展史.

| 范式 | 解决痛点 | 方案 | 局限 |
| --- | --- | --- | --- |
| 符号主义 | 起点 | 物理符号系统 + 规则推理 | 知识瓶颈 + 系统脆弱 |
| 心智社会 | 知识难以形式化, 系统脆弱 | 简单智能体协作 + 涌现 | 缺乏具体实施路径 |
| 学习范式 | 理论美好, 但缺乏实现路径 | 神经网络 + 强化学习 | 依赖海量数据 + 特定语料 |
| LLM 智能体 | 知识与推理难整合, 难适应动态环境 | LLM 为核心 + 工具调用的闭环 | 幻觉 + 工具依赖|

每个新的范式都是为了解决上一范式的核心“痛点”或根本局限.
